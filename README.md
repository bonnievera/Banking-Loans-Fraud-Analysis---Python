# Banking-Loans-Fraud-Analysis---Python
This project analyzes banking data using Python to identify transaction patterns, customer behavior, and potential fraud. The dataset contains banking information including customers, accounts, transactions, and loans.

## Tools Used

Python
Pandas
Matplotlib
Seaborn
Scikit-learn
Jupyter Notebook

## Project Workflow
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis
4. Fraud Detection

## Exploratory Data Analysis
Transaction Distribution
Loan Amount Distribution
Transaction Trend Over Time

## Fraud Detection
Techniques used:
• IQR outlier detection  
• Large transaction flagging  
• Transaction spike detection
